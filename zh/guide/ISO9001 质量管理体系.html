# ISO9001 质量管理体系

# ISO 9000 家族

# 核心标准

  • ISO 9000-2015 《质量管理体系 基础和术语》
  • ISO 9001-2015 《质量管理体系 要求》
  • ISO 9004-2009 《业绩改进指南》
  • ISO 19001-2011 《质量和环境管理体系审核指南》

# ISO9001作用

  • 市场驱动，
    • 满足市场需求获取外部信任
  • 管理驱动

# ISO9001原则

  1. 以顾客为焦点
  2. 领导作用
  3. 全员参与
  4. 过程方法
  5. 关系管理
  6. 询证决策
  7. 改进

  6. 策划
  7. 支持
  8. 运行
  9. 绩效评价
  10. 改进
